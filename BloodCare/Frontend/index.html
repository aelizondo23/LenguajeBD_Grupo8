Bienvenido a BLOODCARE

Este sistema está diseñado para facilitar la gestión integral del banco de sangre del Hospital San Vicente de Paul. Aquí podrá registrar donaciones, gestionar el inventario de componentes sanguíneos, registrar rechazos, consultar reportes y realizar seguimiento de todas las actividades mediante usuarios autenticados. BLOODCARE garantiza trazabilidad, control y seguridad en cada proceso relacionado con la donación y distribución de sangre.
